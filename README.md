## 🚀 Getting Started

Clone, configure, and launch CyberCore in minutes:

```bash
# Clone fractal core
git clone https://github.com/CrisisCore-Systems/CyberCore.git
cd CyberCore

# Optional: Docker for zero-friction
docker build -t cybercore .
docker run -it -p 3000:3000 --env-file .env cybercore

# Or with Node and npm
git checkout main
npm install
cp .env.example .env
npm run dev
```

### Environment Configuration

Your `.env` file should include:

```
SHOPIFY_API_KEY=your_key_here
SHOPIFY_API_SECRET=your_secret_here
CYBERCORE_ECHO_DEPTH=3
AI_ENDPOINT=your_endpoint_here
```
